# 🛡️ Local PDF Studio

A 100% offline, privacy-first PDF manipulation and AI analysis tool built with Python and Streamlit. 

Designed specifically for finance professionals, Chartered Accountants, and auditors who handle highly sensitive client data, this tool bridges the gap between workflow efficiency and strict data privacy. It runs entirely on your local machine—meaning zero data ever leaves your laptop.

## ✨ Features

### 🛠️ 26+ Core PDF Tools
A comprehensive suite divided into 5 specialized groups without any third-party data exposure:
* **Organize PDF:** Merge multiple documents or split pages into independent files.
* **Optimize PDF:** Compress heavy files, repair corrupted documents, and run OCR on scanned papers.
* **Convert PDF:** Seamlessly convert PDFs to editable formats like Word, Excel, and PowerPoint.
* **Edit PDF:** Add customized watermarks (like "CONFIDENTIAL"), insert text, rotate, and crop.
* **PDF Security:** Digitally sign documents, permanently redact text, protect files with passwords, and visually compare PDFs.

### 🤖 PDF Intelligence (Local AI)
Advanced AI capabilities without the cloud risks. Powered by an integrated **Local LLM (LM Studio)** and a local vector database (**ChromaDB**). 
* Ingest huge financial documents and agreements.
* Chat and extract critical data instantly. 
* Works entirely offline, ensuring absolute client confidentiality.
